Comment Ajouter sa Fiche

01
Soumettre

Cliquez sur "+ Ajouter", remplissez le formulaire avec votre nom, race, rang, photo et description. Votre fiche est envoyée en attente.

02
En attente

Votre fiche est stockée dans la base de données avec le statut en_attente. Elle n'est pas encore visible publiquement.

03
Validation admin

Un administrateur Jaharta revoit votre fiche dans le panel admin. Il peut l'approuver, la rejeter ou demander des modifications.

04
Publication

Une fois validée, votre fiche apparaît automatiquement sur le registre, visible par tous les membres de Jaharta.
